# 💼 Portfolio de César Alameda

Portfolio profesional de **César Alameda Barquillo**, desarrollador backend especializado en Java, Spring Boot, PostgreSQL e integraciones ERP.

## 🚀 Stack

- **Framework**: [Next.js 15](https://nextjs.org/) (App Router)
- **Lenguaje**: TypeScript
- **Estilos**: Tailwind CSS 4 con tema glassmorphism
- **Internacionalización**: next-intl (español/inglés)
- **Animaciones**: Framer Motion
- **Contenido**: MDX + JSON
- **Despliegue**: GitHub Pages (estático)
