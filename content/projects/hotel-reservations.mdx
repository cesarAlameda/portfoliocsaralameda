## Descripción

Sistema completo de reservas hoteleras desarrollado como proyecto personal, permitiendo la gestión de habitaciones, reservas y clientes a través de una interfaz web.

## Características

- Gestión de habitaciones y disponibilidad
- Sistema de reservas con selección de fechas
- Panel de administración para gestión de clientes
- Base de datos relacional con PostgreSQL

## Tecnologías

Desarrollado con **Java** y **Spring Boot** en el backend, con **PostgreSQL** como base de datos. Frontend construido con **HTML**, **CSS** y **JavaScript** vanilla para mantener la aplicación ligera y rápida.

## Description

Complete hotel reservation system developed as a personal project, allowing management of rooms, reservations, and clients through a web interface.

## Features

- Room and availability management
- Reservation system with date selection
- Admin panel for client management
- Relational database with PostgreSQL

## Technologies

Built with **Java** and **Spring Boot** on the backend, with **PostgreSQL** as the database. Frontend built with vanilla **HTML**, **CSS**, and **JavaScript** to keep the application lightweight and fast.
