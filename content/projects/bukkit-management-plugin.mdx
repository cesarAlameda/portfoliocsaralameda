## Descripción

Plugin para servidores Minecraft (Bukkit/Spigot) que implementa un sistema completo de gestión de jugadores y configuración dinámica mediante archivos JSON.

## Características

- Sistema de configuración basado en JSON
- Gestión de jugadores con persistencia de datos
- Comandos configurables
- Sistema de eventos y recompensas automáticas
- Optimizado para alto rendimiento

## Arquitectura

Desarrollado en **Java** utilizando la API de **Bukkit/Spigot**, con un sistema de configuración modular basado en JSON que permite personalizar todos los aspectos del plugin sin necesidad de recargar el servidor.

## Description

Plugin for Minecraft servers (Bukkit/Spigot) that implements a complete player management system with dynamic JSON configuration.

## Features

- JSON-based configuration system
- Player management with data persistence
- Configurable commands
- Event system and automatic rewards
- Optimized for high performance

## Architecture

Built in **Java** using the **Bukkit/Spigot** API, with a modular JSON configuration system that allows customizing all aspects of the plugin without server reload.
