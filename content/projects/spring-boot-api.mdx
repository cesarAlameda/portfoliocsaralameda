## Descripción

API RESTful completa construida con Spring Boot, diseñada para manejar operaciones CRUD empresariales con rendimiento y seguridad.

## Características

- Arquitectura RESTful con Spring Boot
- Persistencia con JPA/Hibernate y PostgreSQL
- Pool de conexiones optimizado con HikariCP
- Documentación de API con Swagger/OpenAPI
- Autenticación y autorización
- Pruebas unitarias y de integración

## Tecnologías

Backend completo en **Java** con **Spring Boot**, utilizando **JPA** y **Hibernate** para la capa de persistencia sobre **PostgreSQL**. Conexiones gestionadas con **HikariCP** para rendimiento óptimo.

## Description

Complete RESTful API built with Spring Boot, designed to handle enterprise CRUD operations with performance and security.

## Features

- RESTful architecture with Spring Boot
- Persistence with JPA/Hibernate and PostgreSQL
- Optimized connection pooling with HikariCP
- API documentation with Swagger/OpenAPI
- Authentication and authorization
- Unit and integration tests

## Technologies

Full backend in **Java** with **Spring Boot**, using **JPA** and **Hibernate** for the persistence layer on **PostgreSQL**. Connections managed with **HikariCP** for optimal performance.
